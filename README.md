# My Website Project

A professional multi-page website built with HTML and CSS featuring a modern, responsive design.

## 📁 Project Structure

```
my_website/
├── index.html          # Home page
├── about.html          # About page
├── services.html       # Services page
├── contact.html        # Contact page
├── style.css           # Shared stylesheet
├── assets/             # Images and resources (optional)
└── README.md           # This file
```

## 🎨 Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Multi-Page Navigation** - Easy navigation between pages
- **Modern Styling** - Clean and professional CSS design
- **Smooth Animations** - Hover effects and transitions
- **Contact Form** - Ready-to-use contact form
- **SEO Friendly** - Proper HTML structure for search engines

## 📄 Pages Included

1. **Home (index.html)** - Landing page with featured services
2. **About (about.html)** - Company information and team details
3. **Services (services.html)** - Detailed list of services
4. **Contact (contact.html)** - Contact form and information

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate between pages using the navigation menu

## 🛠️ Customization

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
    /* ... other colors ... */
}
```

### Update Content
Simply edit the HTML files to update:
- Page titles
- Text content
- Links and navigation
- Contact information

### Add Images
1. Create an `assets/images/` folder
2. Add your images there
3. Reference them in HTML:
```html
<img src="assets/images/your-image.jpg" alt="Description">
```

## 📱 Responsive Breakpoints

- Desktop: 1200px and above
- Tablet: 769px - 1199px
- Mobile: Below 768px

## 📝 Tips for Development

- Use meaningful class and ID names
- Keep your HTML structure semantic
- Test on multiple devices
- Optimize images for web
- Consider accessibility (alt text, proper headings)
